# API de Agregação de Dados - Cidades Brasileiras 🇧🇷

Esta é uma API desenvolvida como atividade parcial da disciplina de Técnicas de Integração de Sistemas. O objetivo principal do projeto é agregar e padronizar dados consumidos de APIs externas.

## Tecnologias Utilizadas
* **Linguagem:** Java 21
* **Framework:** Spring Boot

## Pré-requisitos
* Git
* Java Development Kit (JDK) 21

## Como Executar o Projeto

1. Clone o repositório localmente:
   ```bash
   git clone [https://github.com/joaopaulosg21/projeto-faculdade-integracao-de-sistemas.git](https://github.com/joaopaulosg21/projeto-faculdade-integracao-de-sistemas.git)

2. Após isso basta abrir em sua IDE de preferência e executar a partir da **DadosClimaticosApplication.java**
